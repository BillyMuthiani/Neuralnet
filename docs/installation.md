# Installation

## Requirements

- Python 3.10 or higher
- NumPy 1.20 or higher

## Install from PyPI

```bash
pip install kronyx
```

## Install from Source

```bash
git clone https://github.com/Kronyx/kronyx.git
cd kronyx
pip install -e ".[dev]"
```

## Development Installation

```bash
pip install -e ".[dev]"
```

This installs the package in development mode along with development dependencies (pytest, ruff, mypy).
